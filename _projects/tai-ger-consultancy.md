Customer relationship and application workflow portal built from scratch. The stack centers on REST APIs with MongoDB and Node.js, a React SPA with Material-UI, and infrastructure-as-code deployments using **AWS CDK** and CloudFormation.

The main outcome was shortening redundant file and email exchanges among applicants, agents, and editors by centralizing state and approvals in one system.
